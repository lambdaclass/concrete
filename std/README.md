# Concrete Standard Library

This directory contains the current standard library sources.

The detailed design direction for the stdlib lives in:

- [`docs/stdlib/STDLIB_LEADERBOARD.md`](../docs/stdlib/STDLIB_LEADERBOARD.md)
- [`research/stdlib/stdlib-design.md`](../research/stdlib/stdlib-design.md)

That document is the main source of truth for:

- the ranked module build order
- proof-facing formal stdlib modules
- what the stdlib is missing
- which ideas are worth borrowing from other languages
- the recommended order to build the stdlib out
