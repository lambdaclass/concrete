# WC-0013 — if-else as expression

**Category:** error-regression
**Status:** fixed
**Discovered:** historical Bug 008

## Symptom

`let x: i32 = if cond { 10 } else { 20 };` was rejected — `if/else`
was statement-only at the time. Programs that wanted expression-form
`if` had to use a temporary variable + assignment.

## Expected behavior

`tests/programs/bug_if_expression.con` compiles and returns `0`.

## Regression command

`make test-wrong-code`.

## Historical context

Pre-corpus Bug 008. Adding expression-form `if/else` was a language
addition with related codegen support for the merge phi. The
manifest entry guards against accidental removal of the expression
form.
